### Included Extensions

These PHP extensions are enabled:

- `bcmath`
- `curl`
- `gd`
- `iconv`
- `intl`
- `json`
- `mbstring`
- `mcrypt`
- `opcache`
- `pdo`
- `pdo_mysql`
- `redis`
- `soap`
- `xml`
- `xdebug`
- `xsl`
- `zip`
